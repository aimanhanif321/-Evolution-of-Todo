# Register UI Design

## Instructions
Design signup UI with:
1. Name, Email, Password
2. Password strength indicator
3. Terms & conditions checkbox
4. Primary CTA emphasis

## UX Rules
- Do not overwhelm user
- Show validation inline
- Progressive disclosure preferred

## Output
- Form layout
- Validation UX
