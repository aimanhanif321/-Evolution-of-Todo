# Todo Action Buttons

## Instructions
Create buttons for:
- Add Todo
- Complete / Incomplete
- Edit
- Delete (destructive)

## UX Rules
1. Destructive actions must be visually distinct
2. Icons + text preferred
3. Confirm before delete
4. Fast feedback on click

## Output
- Button variants
- Visual hierarchy
- Interaction behavior
