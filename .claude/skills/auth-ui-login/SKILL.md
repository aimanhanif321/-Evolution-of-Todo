# Login UI Design

## Instructions
Design login screens with:
1. Email / Username input
2. Password field with visibility toggle
3. Primary login button
4. Secondary actions (Forgot password, Sign up)
5. Clean, distraction-free layout

## UX Rules
- Minimal fields
- Clear error feedback
- Autofocus first input
- Mobile friendly

## Output
- Layout structure
- Component breakdown
- Interaction behavior
