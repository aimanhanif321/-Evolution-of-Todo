# Modern Card Design

## Instructions
Design cards with:
1. Rounded corners
2. Soft shadows or glass effect
3. Clear padding system
4. Hover feedback
5. Responsive width

## Card Types
- Info card
- Action card
- Status card (Todo)

## Output
- Card structure
- Variants
- Spacing rules
