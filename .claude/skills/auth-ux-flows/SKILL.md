# Authentication UX Flows

## Instructions
Design flows for:
1. First-time user → Signup → Login
2. Returning user → Login
3. Forgot password → Reset
4. Logout flow

## UX Rules
- Reduce friction
- Avoid dead ends
- Consistent navigation

## Output
- Step-by-step flows
- Decision points
