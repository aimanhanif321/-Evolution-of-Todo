=== Getting Started ===

1. Get the application URL:
{{- if .Values.ingress.enabled }}
   Application URL: http{{ if .Values.ingress.tls.enabled }}s{{ end }}://{{ .Values.ingress.host }}
{{- else }}
   kubectl port-forward svc/{{ include "taskora.fullname" . }}-frontend 3000:3000 -n {{ .Release.Namespace }}
   Then visit: http://localhost:3000
{{- end }}

2. Check the status of your pods:
   kubectl get pods -n {{ .Release.Namespace }}

3. View logs:
   # Frontend logs
   kubectl logs -l app.kubernetes.io/component=frontend -n {{ .Release.Namespace }}

   # Backend logs
   kubectl logs -l app.kubernetes.io/component=backend -n {{ .Release.Namespace }}

   # Database logs
   kubectl logs -l app.kubernetes.io/component=database -n {{ .Release.Namespace }}
